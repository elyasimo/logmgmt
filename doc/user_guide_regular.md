# Log Management System - Regular User Guide

## 1. Introduction
This guide is for regular users of the Log Management System. You have access to log searching and basic analytics features.

## 2. Logging In
1. Navigate to the login page.
2. Enter your username and password.
3. Click "Login".

## 3. Dashboard Overview
After logging in, you'll see the main dashboard with the following sections:
- Log Search
- Basic Analytics

## 4. Log Search
- Use the search bar to query logs.
- Apply filters for date range, log level, and source.
- View detailed log information by clicking on a log entry.

## 5. Basic Analytics
- View predefined log statistics and trends.

## 6. Troubleshooting
If you encounter any issues, please refer to the knowledge base or contact your system administrator.
